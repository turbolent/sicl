# sicl

Simple SPARQL library written in Common Lisp.

## Requirements

 * [puri](http://puri.b9.com/)
 * [lexer](http://github.com/turbo24prg/lexer/)
 * [rdf](http://github.com/turbo24prg/rdf/)
 * [cl-yacc](http://www.pps.jussieu.fr/~jch/software/cl-yacc/)
 * [cl-ppcre](http://weitz.de/cl-ppcre/)
 * [cl-interpol](http://weitz.de/cl-interpol/)
